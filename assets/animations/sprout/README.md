# Sprout Animation Assets

This directory contains the source manifest for **Sprout**, the hydroMazing
Smart Garden System assistant character for rpi-clippy.

Sprout is derived from the original Clippy animation set. The individual PNG
frames in `src/renderer/images/animations/sprout/` were generated by recoloring
Clippy's frames with a plant/garden green palette using:

```bash
python3 tools/generate-sprout-assets.py
```

The generated `src/renderer/sprout-animations.tsx` module exports
`SPROUT_ANIMATIONS`, which is wired into the character registry at
`src/renderer/character-animations.tsx`.
